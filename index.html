The OrangeJuice Project

For large scale mobile and Frontend JS aps, there is a need for order and sophistication. OrangeJuice Provides that. For Laravel users, this package is a wrapper and a formatter for backend data to be passed as either arrays or JSON. OrangeJuice takes Laravel object data and builds it into arrays with custom indexes. Then, it will then return the data in a standard format, that never changes so that devs relying on it are never confused. succes, error, cache, data are the bread and butter of OrangeJuice.

Easy to Install

OrangeJuice can be installed easily, by adding the following line to your composer.json file. 

"wiretech/orange-juice": "dev-master"
After this, you will need to add the following lines to your providers and alias arrays respectively in your app/config/app.php file. 
--- providers ---
'Wiretech\OrangeJuice\OrangeJuiceServiceProvider'

--- alias ---
'Citrus'=>'Wiretech\OrangeJuice\Classes\Citrus'
Then, run composer update followed by composer install to install the package

Easily Customizable

OrangeJuice is lightweight and is easily customizable. If you see something you want in it, add it! That's the beauty of it. The code is commented and easy to understand, so anyone can jump right in.
